# Vyshyvanka Editor

Десктопний редактор схем піксельної вишивки у стилі традиційної української вишиванки. При запуску автоматично генерується орнамент з імені автора.

---

## Як користуватись

### Створення проєкту
1. Натисни **Create** у верхньому меню
2. Введи назву (літери, цифри, `-`, `_`)
3. Вибери розмір сітки кнопками `−` / `+` (від 8×8 до 96×96)
4. Натисни **Create**

### Малювання
- **Клік по клітинці** - поставити хрестик активним кольором
- **Eraser** (нижня панель) - режим стирання; повторний клік вимикає
- **Clear Canvas** - очистити все полотно

### Вибір кольору
- **Лівий клік** по кольоровому кружечку - зробити його активним
- **Правий клік** по кружечку - замінити колір через палітру
- **Клік по `+`** - додати новий колір у порожній слот

### Симетрія
Увімкни у лівій панелі перед малюванням:
- **Vertical** - дзеркалить відносно вертикальної осі
- **Horizontal** - дзеркалить відносно горизонтальної осі

Обидві одночасно - повна 4-квадрантна симетрія.

### Дублювання полотна
Наведи курсор на кнопку - побачиш прев'ю. Клікни для підтвердження:
- **Right** - скопіювати ліву половину вправо
- **Left** - скопіювати праву половину вліво
- **Down** - скопіювати верхню половину вниз
- **Up** - скопіювати нижню половину вгору

### Анімація
Натисни **Animate** - побачиш як орнамент «розквітає» від центру. **Stop** - зупинити.

### Збереження
Натисни **Save** у верхньому меню:

| Формат | Особливості |
|---|---|
| `.JSON` | Зберігає проєкт для подальшого редагування |
| `.PNG` | Прозорий або непрозорий фон |
| `.JPG` | Лише непрозорий фон, менший розмір файлу |
| `.GIF` | Статичний або анімований, прозорий або непрозорий фон |

Проєкт також **автозберігається кожні 10 секунд**.

### Відкриття
Натисни **Open**:
- **Browse...** - відкрити `.json` файл з диску
- **Recent Projects** - список останніх збережень (кнопка `×` видаляє запис)
